# Datalab discrete colour scale

Datalab discrete colour scale

## Usage

``` r
scale_colour_datalab_d(..., reverse = FALSE)

scale_color_datalab_d(..., reverse = FALSE)
```

## Arguments

- ...:

  Passed to
  [`ggplot2::discrete_scale()`](https://ggplot2.tidyverse.org/reference/discrete_scale.html).

- reverse:

  Logical. Reverse palette order.
