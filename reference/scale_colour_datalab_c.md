# Datalab continuous colour scale

Datalab continuous colour scale

## Usage

``` r
scale_colour_datalab_c(..., reverse = FALSE, na.value = "grey80")

scale_color_datalab_c(..., reverse = FALSE, na.value = "grey80")
```

## Arguments

- ...:

  Passed to
  [`ggplot2::scale_colour_gradientn()`](https://ggplot2.tidyverse.org/reference/scale_gradient.html).

- reverse:

  Logical. Reverse palette order.

- na.value:

  Colour for missing values.
