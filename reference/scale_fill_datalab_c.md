# Datalab continuous fill scale

Datalab continuous fill scale

## Usage

``` r
scale_fill_datalab_c(..., reverse = FALSE, na.value = "grey80")
```

## Arguments

- ...:

  Passed to
  [`ggplot2::scale_fill_gradientn()`](https://ggplot2.tidyverse.org/reference/scale_gradient.html).

- reverse:

  Logical. Reverse palette order.

- na.value:

  Colour for missing values.
